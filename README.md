## 📌 About DeshSafe

DeshSafe is an open-source web platform that empowers Indian citizens to:

- 📍 **Report disasters and crises** (floods, fires, earthquakes, accidents) with location data
- 🗺️ **View live crisis maps** showing active incidents across India
- 🔔 **Get alerts** for emergencies in their region
- 🤝 **Connect volunteers** with affected communities

The goal: reduce response time and save lives by making crisis information publicly accessible and real-time.

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/Anushka-045/deshsafe.git

# Navigate to the project folder
cd deshsafe

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
